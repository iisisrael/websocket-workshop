Lesson 7 - Experiments with WebRTC (Back home)

Choose Alice or Bob. Alice will join Bob's chat. On loading the experiment Bob will click the 'Create' button and Alice will click the 'Join' button

Load the WebRTC experiment (then come back here)

  1. Alice Enter 'Alice.findBob(bobsIp)' in the browser console where 'bobsIp' is... Bob's IP address. Find this using 'ifconfig' or some other tool.
  2. Bob (On webRTC page) click 'Create', copy the string (it might take a moment to appear) and click "Okay I sent it"
  3. Bob (Back here) Enter 'Bob.startChatWithAlice(connectionString)' in the browser console
  4. Alice Check the console for Bob's connection string. (On webRTC page) enter the string and click "Okay I pasted it"
  5. Alice (On webRTC page) Another string will appear, copy it and click "Okay I sent it"
  6. Alice (Back here) Enter 'Alice.respondToBob(connectionString)' in the console
  7. Bob Check the console for Alice's connection string. (On webRTC page) enter the string and click "Okay I pasted it"

After all these steps both sides should connect. If you have enabled video it should start broadcasting and the text chat will also work. Congratulations, this is a lot easier than actually implementing this for your users!
